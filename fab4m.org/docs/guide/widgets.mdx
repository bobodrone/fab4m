# Component Widgets

A component widget implements the UI for the component.

The widget is part of the component definition, and you can select which
widget to use for each component you add.

All components have a default widget that is used if you don't specify which widget
to use.

## Specifying a widget for a component

A widget can be specified when defining the component in the form, in the example
below we use the select widget for the text field.
